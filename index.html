National Park Service Collection Log in

This is a web application utilizing park information from the National Park Service API. On this home page you will see one hundred parks from the NPS API.
Users have the abiity to add new parks. In order for the park to be sucessfully added users need to provide the following:

  • State
  • URL
  • Weather
  • Name
  • Latitude and Longitude
  • Description
Addtionally, users will have the ability to edit information regarding the park on this page as well.
