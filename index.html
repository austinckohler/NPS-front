National Park Service Information

This is a web application utilizing park information from the National Park Service API. On this home page users will see one hundred parks from the NPS API. Users have the abiity to create an account or login. On their profile page they have the ability to store parks they have visited. Users of this application are not required to create and account or login. Addtionally, users can add new parks. In order for the park to be sucessfully added users need to provide the following:

  • State
  • URL
  • Weather
  • Name
  • Latitude and Longitude
  • Description

Addtionally, users will have the ability to edit information regarding the park on this page as well.
